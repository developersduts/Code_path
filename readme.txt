Під час виконання курсової роботи мені потрібно із використанням JavaFX розробити програму оброблення даних з існуючих файлів та занесення оброблених даних у файли, створені засобами JAVA.
1.	Формат існуючих файлів (футболісти):
Footballers   Country    Age
  Messi        Argentina   28
…………………………….
2.	Кількість існуючих файлів – 3;
3.	Зміст оброблення даних з файлів:
-	Утворити файл, до якого занести футболістів віком, не більшим за 22;
-	Утворити файл, до якого занести футболістів віком, в межах від 23 до 27;
-	Утворити файл, до якого занести футболістів віком, більше за 27;
4.	Сортування у вихідних файлах проводити по трьох варіантах:
-	По Footballers, при рівності - по Country, при рівності – по Age;
-	По Age, при рівності - по Footballers, при рівності – по Country;
-	По Country, при рівності - по Age, при рівності – по Footballers;
